# Design System (DESIGN.md)

This file contains the design tokens extracted from the Dribbble reference (19377958) and the UI/UX Design Brief.

## Colors
- **Canvas / Background (`bg-canvas`)**: `#F7F3EE` (Warm off-white/cream)
- **Sidebar (`bg-sidebar`)**: `#FFFFFF`
- **Insight Panel (`bg-panel-accent`)**: `#C3DEDD` (Powder teal)
- **Primary Accent (`accent-primary`)**: `#EF5B4B` (Coral)
- **Primary Text (`text-primary`)**: `#17181C`
- **Secondary Text (`text-secondary`)**: `#9B9B9B`
- **Positive Status (`status-positive`)**: `#3FAE71`
- **Warning Status (`status-warning`)**: `#E0A020`
- **Illustration Warmth (`illustration-warm`)**: `#F5DCC0`
- **Card Background (`card-bg`)**: `#FFFFFF`

## Typography
- **Display Numerals / Primary Headings (Comico)**: Bold, heavy rounded text.
- **Section Labels / Brand (Zodiak)**: Editorial serif.
- **Body / Chrome (Sans-serif e.g., Inter)**: Clean legibility for small text.

## Spacing & Radius
- **Card Border Radius**: 24px (`2xl` or `3xl` in Tailwind)
- **Pill / Button Radius**: 9999px (`full`)
- **Card Inner Padding**: 24px
- **Gaps / Gutters**: 16px - 24px
